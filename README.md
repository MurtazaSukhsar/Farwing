# FARWINGS TECH SOLUTIONS

A premium, futuristic website for FARWINGS TECH SOLUTIONS — built with cutting-edge web technologies.

## Features

- **3D Particle Background** — Interactive Three.js particle system with floating elements
- **Glassmorphism Design** — Premium frosted glass UI components throughout
- **Framer Motion Animations** — Smooth scroll-triggered animations and transitions
- **GSAP-Ready** — Prepared for advanced scroll animations
- **Responsive Design** — Fully optimized for mobile, tablet, and desktop
- **Loading Screen** — Cinematic preloader with progress animation
- **Cursor Glow Effect** — Mouse-reactive ambient lighting
- **Animated Stats Counters** — Number count-up animations on scroll
- **Testimonial Slider** — Auto-rotating client testimonials
- **Interactive Contact Form** — Floating form fields with neon glow effects

## Tech Stack

- React 19
- Tailwind CSS 3.4
- Framer Motion
- Three.js / React Three Fiber
- GSAP (configured)
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
